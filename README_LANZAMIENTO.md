## 📞 SI ALGO NO FUNCIONA

### 🔴 Problema: RLS SQL da error en Supabase

**Solución:**
- Verifica que NO hay otro SQL ejecutándose
- Copia EXACTAMENTE el archivo (sin modificar)
- Intenta nuevamente

### 🔴 Problema: Sentry no reporta crashes

**Solución:**
- Verifica que DSN NO es `'YOUR_SENTRY_DSN'`
- npm install @sentry/react-native @sentry/expo
- Descomentar en App.tsx
- Reiniciar app

### 🔴 Problema: Stripe/SendGrid no funciona

**Solución:**
- Verifica API keys correctas
- Verifica que estén en production, no en test mode
- Lee documentación oficial de servicio

### 🔴 Problema: App Store rechaza app

**Solución:**
- Verifica que Privacy Policy URL es válida
- Verifica que tiene mínimo 2 screenshots
- Verifica ESRB rating completado
- Espera 24h y intenta nuevamente
